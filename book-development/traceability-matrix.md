# Traceability Matrix

This document ensures that every part, chapter, and draft supports the book's core thesis.

## Core thesis

Software projects fail primarily through specification drift: the delivered software diverges from the intended software.

Intentional Development is a framework for preserving intent from domain understanding to deployed features through shared intent, behavior, and verification.
